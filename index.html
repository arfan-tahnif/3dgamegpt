Combat Arena Web

A lightweight HTML5 canvas survival shooter built with plain HTML, CSS, and JavaScript.

Keyboard: WASD / Arrow keys · Mouse: aim and shoot · R: reload · P/Esc: pause

Original browser implementation. No proprietary JAR code or assets included.

How to Play

  • Move with WASD or Arrow Keys.
  • Aim with your mouse or touch direction.
  • Click/tap to shoot. Press R to reload.
  • Survive waves, collect medkits and ammo packs.
